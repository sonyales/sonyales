# Hi, I'm Sonya 👋

Frontend Developer building clean, responsive and maintainable web interfaces.

I work with React, TypeScript, Redux Toolkit and modern frontend tooling. I build SPA applications with REST API integration, routing, authentication, forms, state management and tested user scenarios.

## About me

- Build interfaces from individual UI components to full SPA applications
- Think through component structure, application state and user flows
- Care about readable code, reusable components and long-term project maintainability
- Practice testing of key logic and important user flows
- Interested in frontend architecture, accessibility and thoughtful user experience
